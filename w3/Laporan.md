#Ketentuan pelaporan w3
Jawab juga pertanyaan analisis berikut:
1. Gen apa saja yang mengalami upregulation dan downregulation, ditampilkan dalam bentuk volcano plot.
2. 50 Differentially Expressed Genes (DEGs) teratas, ditampilkan dalam bentuk heatmap.
3. Analisis enrichment, yang mencakup:
- Gene Ontology (GO)
- KEGG Pathway: Hasil analisis enrichment wajib disertai visualisasi/plot yang relevan.

B. Format Laporan yang Dikumpulkan
Peserta dapat menyusun laporan singkat menggunakan Bahasa Indonesia atau Bahasa Inggris, dengan struktur sebagai berikut:
1. Pendahuluan
2. Metode
3. Hasil dan Interpretasi
4. Kesimpulan

C. Ketentuan Pengumpulan
1. Laporan dikumpulkan dalam format PDF atau Markdown (.md) (format Markdown lebih disarankan).
2. Seluruh hasil tugas wajib diunggah ke repository GitHub pribadi masing-masing peserta.
3. Repository GitHub harus memuat komponen berikut:
- Laporan (PDF atau Markdown)
- Script analisis yang digunakan
- Dataset yang dianalisis
- Seluruh plot/visualisasi hasil analisis (volcano plot, heatmap, enrichment plot, dan lainnya)
4. Tautan repository GitHub dicantumkan pada kolom pengumpulan di Google Classroom.
5. Tenggat waktu pengumpulan: Minggu, 22 Februari 2026
